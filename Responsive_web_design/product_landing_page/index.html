Installing python

If you haven't downloaded an installer yet, follow these steps:

  • Go to the downloads page of Python.
  • Select the version you want to use
  • Click on the download link.

Once your download is finished, do the following:

  • Open your download location.
  • Double click on the installer.
  • Follow the installation steps as prompted.

You should have Python installed on your system now!

Set up an IDE for python

You can use any integrated development environment (IDE), which is a software application that helps programmers develop software code efficiently. I prefer using Visual Studio Code (vsCode), but other options like JetBrains PyCharm are also available.

You can download the IDE from the respective websites linked above, and set up your environment. For example, for vsCode, you would install extensions for python and jupyter notebook in the extensions tab, allowing you to create, edit and run .py (Python) and .ipynb (Python Notebook) files in the IDE.

You can also select an interpreter from your IDE, which allows you to seamlessly switch between different Python versions if you want to. For example, you have code developed in python3.10 that you need to test, but your interpreter is set to python3.12, then you can just switch it in the IDE to allow you to test the older code (assuming that the required packages are installed).

Let's get you started

This section will be written as if you are using vsCode, you have python3.10 installed, and you have the Python and Jupyter Notebook extensions installed in the IDE.

For basic Python code, you can open your IDE, and create a new file called run.py. You can enter the following code into it:

print("Hello World")

You can save the file and run it by either pressing the "play" button, or by executing the following in the IDE native command line interface:

python3.10 -m run.py

If you execute the code as is it should print "Hello World" in the command line interface.
